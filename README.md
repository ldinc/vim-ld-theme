# vim-ld-theme
Simple light theme for vim

## Warning
* Still pretty works only on 256-term!

## Bundle install:
In your .vimrc add:
```
Plugin 'ldinc/vim-ld-theme'
```
and then use command in vim:
```vim
:PluginInstall
```

## Common install:
Copy ``` colors/ld.vim ``` into your vim folder (by default: ``` ~/.vim/```).

## Using:
```vim
colorscheme ld
```
